RDL Touch – Website README
Overview
RDL Touch is a modern, responsive website designed for a braiding and beauty service brand.
The site showcases services, pricing, booking options, and a gallery featuring TikTok-embedded videos and photos. It also includes bilingual navigation (English/French) and a light/dark theme toggle.
This README explains the structure, features, and setup of the project.

Features
Multi‑Language Support
- Language switcher at the top of the page
- Links to English (Doc.html) and French (Doc-fr.html) versions
   Branding & Layout
- Custom logo displayed in the header
- Clean navigation bar with links to all main pages:
- Home
- Services
- Book Now
- Prices
- Gallery
- Contact Us
- About Us
 Services Section
Highlights the main braiding services offered:
- Box Braids
- Twists
- Feeding Cornrows
- Weaves
- Boho Styles
   TikTok Gallery Integration
The gallery includes embedded TikTok videos and photo posts using:
<script async src="https://www.tiktok.com/embed.js"></script>
Booking Section
A call‑to‑action button linking to the booking page.
 Theme Toggle
A button in the footer allows users to switch between light and dark themes using JavaScript (Toggle.js).
 Footer
Includes:
- Copyright
- Privacy Policy
- Terms of Service
- Social media links (TikTok)
